# 📸 Screenshots

Drop your screenshots in **this folder** using the exact file names below, and they
will appear automatically in the main [`README.md`](../../README.md).

> **Tip:** PNG works best. Use a clean browser window (no bookmarks bar / dev tools),
> a logged-in account with a few ratings so the recommendations look alive, and a
> consistent width (~1400px) so all images line up nicely.

## Capture checklist

| File name | Page / URL | What it should show |
|---|---|---|
| `home.png` | `/` | The hero banner + the "Picked for you" recommendations strip |
| `places.png` | `/places/` | The catalogue with star ratings and the live recs strip |
| `model.png` | `/model/` | The algorithm comparison cards with their metrics |
| `admin-dashboard.png` | `/admin/` | The KPI cards and Chart.js charts |
| `admin-recommender.png` | `/admin/` → Recommender | The algorithm cards with the "Apply" control |
| `admin-attractions.png` | `/admin/` → Attractions | A CRUD list with search / filters / export |

## How to take a screenshot

- **Windows:** `Win + Shift + S` (Snipping Tool), then save as PNG here.
- **macOS:** `Cmd + Shift + 4`, then move the `.png` into this folder.
- **Full-page (best):** in Chrome/Edge, open DevTools (`F12`) →
  `Ctrl/Cmd + Shift + P` → type *"Capture full size screenshot"*.
