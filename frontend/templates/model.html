Why this model?

The currently selected engine is {{ model_name }}. Interestingly, the simple baseline (just the average rating of a place plus how generously a user rates) often scores the lowest RMSE on this dataset — there is limited collaborative “signal” in a small set of ratings.

But the baseline gives every user the same ranking, so it cannot personalise. Since the whole point of the app is recommendations that react to your ratings, we use the best personalised model instead. This is a useful reminder that RMSE measures rating prediction, not ranking quality — and the two can disagree.
